# AI-Cover-Letter-Generator
AI Cover Letter Generator is a web-based application that uses Google Gemini AI to generate personalized, ATS-friendly cover letters from user inputs and uploaded resumes. It automatically extracts resume details, tailors content to the target job role and company, and allows users to copy or download the generated cover letter as a professionally formatted PDF

Features
🤖 AI-powered cover letter generation using Google Gemini
📄 Resume PDF upload
🎯 Personalized content based on job role and company
✅ ATS-friendly formatting
📋 One-click copy to clipboard
📥 Download cover letter as PDF
🎨 Clean, responsive, and modern UI

Tech Stack
HTML5
CSS3
JavaScript
Google Gemini API
PDF.js

Working Site URL: https://ai-cover-letter-generator-rosy-nu.vercel.app/
